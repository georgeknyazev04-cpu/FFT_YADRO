# FFT_YADRO
Класс для БПФ, разработанный в рамках выполнения технического задания YADRO.
Был реализован классический алгоритм Кули-Тьюки, случай n = 2 вычисляется обычной "бабочкой", с оптимизацими вычислений БПФ для n = 3,5, при помощи алгоритма Винограда (thanks to https://ru.dsplib.org/content/fft_winograd/fft_winograd.html).
Для длин преобразований, не факторизуемых по степеням 2,3,5 было реализовано классическое "медленное" ДПФ.
